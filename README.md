# 🏭 SmartFind — 스마트공장 공급기업 수집기

[![Streamlit](https://static.streamlit.io/badges/streamlit_badge_black_white.svg)](https://smartfind.streamlit.app)

**smart-factory.kr** 공급기업 데이터를 자동 수집하고, 필터링하여 엑셀로 추출하는 도구입니다.

---

## 주요 기능

| 기능 | 설명 |
|------|------|
| **원클릭 수집** | 버튼 하나로 전체 3,300+건 공급기업 데이터 수집 (약 6분) |
| **필터 체크박스** | 구분(제조/서비스/공방), 전문분야(17개), 특화업종(24개) 체크 후 추출 |
| **엑셀 다운로드** | 네이비 헤더, 줄무늬, 색상 구분, ★만족도 등 고가독성 엑셀 출력 |
| **드래그앤드롭 병합** | 이전 엑셀 업로드 → 신규/변경분만 자동 추가 (중복 제거) |

## 수집 항목

| 필드 | 설명 |
|------|------|
| 업체명 | 공급기업 이름 |
| 공급기업 구분 | 스마트제조 / 스마트서비스 / 스마트공방 |
| 구축건수 | 스마트공장 구축 실적 |
| 지역 / 주소 | 소재지 |
| 종사자규모 | 직원수 |
| 매출규모(억) | 연간 매출 |
| 만족도(5점) | 수요기업 만족도 평가 |
| 주력분야 / 전문분야 | ERP, MES, AI 등 기술 분류 |
| 특화업종 | 한국 산업분류 코드 기반 |
| 대표자 / 설립일 / 전화 / 홈페이지 / 사업자번호 | 기본 정보 |

## 필터 구조

```
■ 공급기업 구분
  전체 | 스마트제조 | 스마트서비스 | 스마트공방

■ 전문분야
  자동화기기    : 산업용로봇, 스마트장비, 식별시스템/머신비전
  연결화기기    : 제어시스템/컨트롤러, 센서/액추에이터, 통신네트워크장비
  정보화솔루션  : ERP, PLM/PDM, SCM/WMS, MES/POP/APS, FEMS, QMS/CMMS
  지능형서비스  : 클라우드컴퓨팅, CPS/디지털트윈, 제조빅데이터/제조AI, AR/VR/MR, 컨설팅/사이버보안

■ 특화업종 (한국 산업분류 코드)
  식료품, 섬유, 화학, 금속, 전자, 기계, 자동차 등 24개 업종
```

## 로컬 실행

```bash
python -m venv venv
venv\Scripts\pip install -r requirements.txt
venv\Scripts\streamlit run streamlit_app.py
```

## 기술 스택

- **API**: smart-factory.kr REST API (리버스 엔지니어링)
- **프론트엔드**: Streamlit
- **엑셀 생성**: openpyxl (스타일링 포함)
- **배포**: Streamlit Cloud (무료)
